Features
  • Live reloading with browser-sync on 3000 port,
  • SCSS to CSS transpilation, autoprefixing, optimization, and minification,
  • ES6+ to ES5 transpilation and minification,
  • Write your own HTML modules and import them to pages located in src folder, pass 'props' to modules and use expressions
Usage
  • Write your HTML pages in / folder
  • Write your SCSS code in src/styles folder
  • Write your JS code in src/js folder
  • Write your HTML modules in src/modules folder
Available commands
  • npm start - command starts your project. SCSS is parsing to CSS, JS isn't minified and HTML is compiled,
  • npm run build:all - command builds our application.
    • CSS is minified, transpiled and optimazed,
    • JS is minified and transpiled,
    • HTML is compiled,
